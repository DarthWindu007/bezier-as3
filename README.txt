Features:

     Image-01. - Teapot Uniform Tessellation
     Command Line: ./as3 teapot.bez 0.1 
     Features: Tessellate the patch using uniform technique with 0.1 step size.
     

     Image-02. - Smooth Teapot Uniform Tessellation 
     Command Line: ./as3 teapot.bez 0.1 (press s)
     Features: Tessellate the patch using uniform technique with 0.1 step size, toggle between flat and smooth shading. 
     

     Image-03. - Wireframe Teapot Uniform Tessellation
     Command Line: ./as3 teapot.bez 0.1 (press w) 
     Features: Tessellate the patch using uniform technique with 0.1 step size, wireframe mode
     


     Image-04. - Hidden-Filled Line Teapot Uniform Tessellation (Optional Feature)
     Command Line: ./as3 teapot.bez 0.1 (press h) 
     Features: Tessellate the patch using uniform technique with 0.1 step size, toggle between filled and hidden-line mode
     
     

     Image-05. - Teapot Adaptive Tessellation
     Command Line: ./as3 teapot.bez 0.1 -a 
     Features: Tessellate the patch using adaptive technique with 0.1 step size.
